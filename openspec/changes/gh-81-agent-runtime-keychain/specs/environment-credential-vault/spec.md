## ADDED Requirements

### Requirement: Wizard credentials use OS-backed encryption
Credential material captured or generated by the wizard, including SSH key passphrases and daemon session tokens, SHALL be encrypted through Electron `safeStorage` in the main process before persistence and SHALL never be stored in plaintext.

#### Scenario: SSH key passphrase is supplied
- **WHEN** the user completes SSH setup with a key passphrase
- **THEN** the main process stores only an OS-encrypted credential payload outside the environment config

#### Scenario: Pairing returns a session token
- **WHEN** daemon pairing succeeds during the wizard
- **THEN** the main process stores the access token as an OS-encrypted credential payload

#### Scenario: OS encryption is unavailable
- **WHEN** the wizard needs to persist credential material and `safeStorage` is unavailable
- **THEN** setup fails without storing the credential in plaintext

### Requirement: Environment config stores credential references only
Environment configuration SHALL contain only opaque references to credentials and SHALL NOT contain passphrases, access tokens, or encrypted credential payloads.

#### Scenario: Environment with wizard credentials is read
- **WHEN** the environment configuration is loaded or sent to the renderer
- **THEN** it contains credential reference identifiers but no credential values or ciphertext

### Requirement: Removing an environment removes its credentials
The system SHALL delete every credential vault entry owned by an environment when that environment is removed.

#### Scenario: Environment is removed
- **WHEN** the user removes an environment with passphrase or session-token references
- **THEN** the referenced vault entries and legacy token entry are deleted before removal completes

### Requirement: Mock mode does not persist secrets
Browser mock mode SHALL support runtime selection without persisting credential material.

#### Scenario: Wizard runs without Electron
- **WHEN** the mock wizard receives a runtime and optional credential input
- **THEN** it returns an environment with the selected runtime and discards credential material
